#####Opção 1.#####
Fiz uso "display: inline-block;" para alinhar lado a lado.
Não usei "border", usei:
	     box-shadow: 0px 0px 1em grey;
            -webkit-box-shadow: 0px 0px 1em #666;
            -moz-box-shadow: 0px 0px 1em #666;
Assim deixando uma sombra mais clara como no objetivo.


#####Opção 2.#####
Fiz uso do "float: left;" para alinhar lado a lado.
Aqui usei "border", desta forma:
	    border: solid 1px grey;
            box-shadow: 0 4px 20px 0 black;
Assim dando um efeito mais 3D, particularmente gostei muito mais.
